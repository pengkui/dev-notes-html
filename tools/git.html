A Typical Scenario: the Modified-View-Stage-Commit workflow


    echo "Assume changes were made on some files"
    vim oldfile1
    vim oldfile2
    vim oldfile3
    
    echo "Display changes (both staged and not-yet-staged) since the previous commit"
    git status
    
    echo "Display not-yet-staged changes line-by-line"
    git diff
    
    echo "Display staged changes line-by-line"
    git diff --staged
    
    echo "Stage changes for commit (i.e. add modified-but-not-yet-staged files into the staging area)"
    git add oldfile1 oldfile2
    
    echo "Recursively add/stage all files under the current working dir"
    git add .
    
    echo "Commit all staged changes"
    git commit -m "comments"
    
    echo "Use the -a switch to auto stage all modified-but-not-yet-staged changes, and then commit"
    git commmit -a -m "comments"
